# 封面生成器

## 目录结构

```
长篇/奇幻/诡异熔炉/
├── 作品信息/           ← 封面 + 作品信息.md 统一在此
│   ├── 作品信息.md     ← 书名、笔名、候选书名、简介
│   ├── cover.png       ← 主书名封面
│   ├── cover-熔炉觉醒.png
│   └── ...
├── 大纲/
├── 设定/
├── 正文/
└── ...
```

## 用法

```bash
# 批量：为所有候选书名各生成一张封面
node <skill_dir>/generate_cover.js "长篇/奇幻/诡异熔炉"

# 单张：为自定义书名单独生成
node <skill_dir>/generate_cover.js "长篇/奇幻/诡异熔炉" --title "开局流放矿坑，我觉醒了分解熔炉"
```

## 作品信息.md 规范

创作时机：设定全部写完 + 至少列完第一卷大纲后。

```markdown
# 作品信息

**作品名称**：诡异熔炉

**笔名**：里克修斯

**候选书名**（至少6个，15字以内）：
1. 诡异熔炉
2. 熔炉觉醒
3. 诡异工业帝国
4. 废土熔炉
5. 熔炉之主
6. 诡异分解者

**作品简介**：
诡异降临的废土世界，少年楚渊觉醒"熔炉"将诡异分解为工业原材料...
```

## 发给 AI 的 prompt

```
请为我的小说生成一个网文封面。
小说标题：《熔炉觉醒》
作者：里克修斯
故事简介：诡异降临的废土世界...
候选书名：
1. 《诡异熔炉》
2. 《熔炉觉醒》
3. 《诡异工业帝国》
...

要求：
1. 封面中清晰显示作品名称和作者笔名，笔名格式为"里克修斯　〇　著"
2. 画面元素符合小说题材和风格
3. 封面设计美观有艺术感
4. 尺寸 600×800 像素，格式 jpg 或 png，文件大小不超过 5MB
```

## 输出

全部写入项目下的 `作品信息/` 目录：

| 场景 | 文件名 |
|------|--------|
| 主书名（作品名称） | `cover.png` |
| 其他候选书名 | `cover-{书名}.png` |
| `--title` 单独调用 | `cover-{书名}.png` |

全部 600×800 像素，PNG 格式。
