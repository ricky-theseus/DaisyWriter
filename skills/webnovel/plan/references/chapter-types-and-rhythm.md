# 章节类型与节奏控制

> 从 chinese-novelist 提取的章节规划参考。plan skill 在 Step 6 按需加载。

## 四类章节

### 1. 情节推进章
- **目的：** 推动主线剧情发展
- **特征：** 有明确的事件进展、重要信息揭示
- **示例：** 主角发现线索、敌人发动攻击、盟友背叛

### 2. 人物深化章
- **目的：** 深化读者对人物的理解
- **特征：** 揭示人物背景、动机、内心冲突
- **注意：** 必须与主线相关，不能是纯粹的人物小传

### 3. 氛围营造章
- **目的：** 建立特定情绪或紧张感
- **特征：** 注重感官描写、节奏控制
- **示例：** 暴风雨前的宁静、潜伏行动、等待审判

### 4. 过渡衔接章
- **目的：** 连接两个重大事件
- **特征：** 信息整理、位置转换、时间跳跃
- **注意：** 保持简洁，避免拖沓；即使过渡也要有一个不可删除的小事件

## 十种强力开头技巧（×5 题材）

### 1. 行动中开场（In Media Res）
直接从冲突或动作的高潮点开始。
- **悬疑/犯罪：** 子弹擦过他的耳边，击碎了身后的花瓶。
- **言情/都市：** 婚纱的拉链卡住了，门外婚礼进行曲已经响起。
- **奇幻/玄幻：** 护山大阵碎裂的瞬间，苏衍知道自己被同门卖了。
- **历史/古风：** 刀光闪过，钦差的人头滚落在青石板上。
- **科幻：** 警报响了。林薇从冬眠舱里醒来，发现飞船上其他舱位全部显示"生命体征消失"。

### 2. 反常情境
呈现一个不符合常理的场景。
- **悬疑/犯罪：** 死人坐在办公桌前，正在写一份报告。
- **言情/都市：** 他对着镜子里自己的脸拨了一个电话。接电话的人说："你打错了。"
- **奇幻/玄幻：** 全城的花在同一秒枯萎。街上老人跪地痛哭，小孩拍手欢笑。
- **历史/古风：** 太阳从西边升起的那天，史官烧掉了所有的史书。
- **科幻：** 测谎仪显示她在说真话。但她说的事情，不可能发生在这个宇宙。

### 3. 震撼对话
用一句惊人的对话开场。

### 4. 倒计时开场
从时间压力开始：炸弹倒计时、登机口还有十分钟关闭、天雷即将劈下。

### 5. 重大发现
从发现关键线索或真相开始：法医报告、丈夫外套里的照片、秘境深处的镜子。

### 6. 危机时刻
从角色面临最大危机开始：门被踹开、元婴碎裂、围城粮草已尽。

### 7. 谜团浮现
从无法解释的现象开始：手背上的陌生纹身、凌晨三点七分的空白短信、枯井突然满水。

### 8. 背叛开场
从背叛或信任崩塌开始：搭档的枪口对准后背、结婚当天收到分手消息、大师兄的剑刺进心口。

### 9. 重大选择
从角色的艰难决定开始：救生艇只能载一个人、升职通知和手术报告同时出现、降可保百姓死则名垂青史。

### 10. 结局预告
从未来的某个关键时刻开始，然后倒叙。

## 节奏模式参考

### 悬疑/动作
```
紧张 → 短暂缓解 → 更紧张 → 高潮 → 悬念结尾
```

### 言情
```
日常互动（轻松）→ 情感波动（升温）→ 误会或阻碍（紧张）→ 真相或告白（高潮）→ 新悬念
```

### 奇幻/玄幻
```
修炼/准备（低密度）→ 危机降临（中密度）→ 战斗或突破（高密度）→ 战后消化（低密度）→ 新挑战浮现
```

### 信息密度波浪
```
高密度 → 低密度 → 高密度 → 低密度
（动作/对话/揭示）→（沉淀/描写/内心）→（新冲突/转折）→（情感消化/环境过渡）
```

操作要点：
- 一段高密度之后必须跟一段低密度
- 连续三段以上高密度 = 读者疲劳
- 连续三段以上低密度 = 读者无聊
- 每章应有 2-3 个密度波浪

## 章节长度与内容密度

| 章节类型 | 字数范围 | 核心事件 | 场景数 | 适用场景 |
|---------|---------|---------|-------|---------|
| 短章节 | 2000-3000 | 1 个主要事件 | 1-2 | 过渡衔接、单一场景聚焦 |
| 标准章节 | 3000-5000 | 2-3 个主要事件 | 2-4 | 大多数章节 |
| 长章节 | 5000-7000 | 3-4 个主要事件 | 3-5 | 高潮、大战、多线收束 |

## 章内段落节奏

### 长短句交替
- 连续三句长度相同就必须打破
- 动作段落：句子控制在 10 字以内
- 转折时刻：用一个极短句制造停顿（"他愣住了。""没有。"）

### 段落呼吸
- **紧张场景：** 一段一两句话，甚至一句话一段
- **缓和场景：** 一段五六句话，允许展开
- **转折爆发：** 长段落中间突然插入一个单句段落

### 全章节奏检查
1. 标记每段的"速度"（快/中/慢）
2. 画一条节奏曲线（视觉化）
3. 如果曲线是一条直线（全程匀速），需要调整
4. 如果全程是快节奏，加入喘息空间
5. 如果全程慢节奏，找到可以加速的冲突点
