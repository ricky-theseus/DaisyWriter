# 章节结构与内容密度指南

> 从 chinese-novelist 提取的章节写作参考。write skill 在 Step 2 起草前按需加载。

## 标准章节结构

### 1. 开头钩子（前 20%）— 最关键

读者在前 20% 决定是否继续阅读。

**必须达到的效果：**
- 即时紧张感 — 读者立即感受到危机或冲突
- 重大事件 — 发生推动剧情的重要事情
- 情感冲击 — 读者产生强烈情绪（好奇、震惊、担忧）
- 继续阅读的欲望 — 迫切想知道接下来发生什么

**开头致命错误（绝对避免）：**

| 错误类型 | 示例 | 为什么致命 |
|---------|------|-----------|
| 天气描写 | "那天天气晴朗，万里无云……" | 与故事无关 |
| 日常流程 | "李明醒来，刷牙洗脸，吃早餐……" | 无聊，没有冲突 |
| 回顾上章 | "上一章我们说到……" | 读者已经知道 |
| 缓慢铺垫 | "先介绍一下这个城市的背景……" | 信息倾倒 |
| 平淡对话 | "你好，你好吗？我很好。" | 无张力 |
| 过度解释 | "这是因为，所以，然后……" | 讲述而非展示 |

### 2. 发展推进（中间 50%-60%）

本章的核心内容，必须推进剧情或深化人物。

**推进方式：**
- **新信息揭示** — 让读者或主角获得重要信息
- **关系变化** — 人物关系发生转变
- **问题升级** — 现状恶化，新危机出现
- **角色成长** — 主角获得新技能、新认知

**避免：**
- 纯粹的场景描写（风景、房间布局等与剧情无关的）
- 与剧情无关的人物互动
- 重复已知的对话
- 用大段内心独白代替行动

### 3. 高潮时刻（后 15%-20%）

本章的情感或动作最高点。

**高潮类型：**
- **动作高潮** — 战斗、追逐、对抗
- **情感高潮** — 重大发现、背叛、告白、牺牲
- **心理高潮** — 主角的内心转折点

### 4. 结尾钩子（最后 5%-10%）

留下悬念，让读者想看下一章。不得使用章末总结体。

## 写作检查清单

每章写完后逐项自查。任何一项不合格就修改。

### 开头检查
- [ ] 前 20% 是否极其吸引人？（如果不是，重写）
- [ ] 是否在第一段就建立冲突或紧张？
- [ ] 是否有重大事件或信息揭示？
- [ ] 是否有强烈的情感冲击？
- [ ] 读者是否会迫切想知道接下来发生什么？
- [ ] 是否避免了所有"致命错误"？（天气、日常、回顾等）

### 内容检查
- [ ] 本章是否推进了主线剧情或深化了人物？
- [ ] 是否有冲突或转折？
- [ ] 是否有核心事件（删除它会影响理解）？
- [ ] 对话是否推动情节或揭示人物？
- [ ] 是否展示了而非讲述了关键信息？
- [ ] 结尾是否留下悬念钩子？
- [ ] 是否为下一章埋下伏笔？

### 人物检查
- [ ] 人物行为符合已建立的性格？
- [ ] 人物对事件有情绪和行动反应？
- [ ] 对话能区分不同角色？
- [ ] 通过行动和对话表现性格，而非直接陈述？

### 对话检查
- [ ] 每句对话有目的（推进情节、揭示人物、制造冲突）
- [ ] 没有"你好""吃了吗"等无意义对话
- [ ] 对话简洁自然，符合真实说话方式
- [ ] 有潜台词，不是所有话都直说
- [ ] 对话标签使用得当，不过度使用副词

### 悬念检查
- [ ] 结尾有钩子，让读者想看下一章？
- [ ] 悬念强度与故事位置匹配？
- [ ] 不是虚假悬念（机械误会、无意义的"突然"）？
- [ ] 为下一章设置了冲突或期待？

### 展示而非讲述检查

| 讲述（避免） | 展示（使用） |
|-------------|-------------|
| 他很愤怒 | 他握紧拳头，指节发白 |
| 她很美丽 | 他凝视着她，忘记说话 |
| 他很紧张 | 他反复调整领带 |
| 房间很乱 | 衣服扔在沙发上，外卖盒堆在桌上 |
| 他很富有 | 他从口袋里掏出一叠现金 |
| 她很伤心 | 她低头搅咖啡，勺子停了很久 |

### 节奏检查
- [ ] 句子长度有变化，没有连续三句长度相同
- [ ] 段落长度有变化，动作场景用短段落
- [ ] 信息密度有高低交替
- [ ] 全章节奏不是一条直线（有波浪感）

### 连贯性检查
- [ ] 与前文连贯，上一章的悬念有回应
- [ ] 伏笔有呼应或即将揭示
- [ ] 时间线一致，事件顺序正确
- [ ] 人物已知信息一致

### 开头自测
如果对开头有以下任何问题回答"否"，必须重写：
1. 读者读了前三段后，会想继续读吗？
2. 开头是否有冲突或危机？
3. 开头是否有意外或转折？
4. 开头是否让读者产生强烈情绪？
5. 开头是否避免了平淡的日常、天气、背景说明？

## 内容密度波浪

写草稿时采取"先放后收"策略：
1. **第一遍：** 只写主干事件和对话，不修饰、不描写环境
2. **第二遍：** 在每个事件之间加入过渡、情绪反应和感官细节
3. **第三遍：** 读一遍，标记节奏太平或太紧的地方，对应加减

## 打破读者预期的四种技术

### 预期反转
建立读者对 A 走向的预期，然后转向 B。铺垫至少 2-3 个段落，反转不能凭空出现。

### 信息差利用
让读者知道角色不知道的事（制造焦虑），或让角色知道读者不知道的事（制造悬念）。信息差持续不超过 2-3 章。

### 复杂动机
没有纯粹的善恶。当读者理解了反派的立场，故事立刻有深度。揭示时机要在读者已经形成判断之后。

### 节外生枝
在看似已解决的问题上引入新变量。新变量必须有逻辑关联，引入时机在读者刚放松警惕的瞬间。
