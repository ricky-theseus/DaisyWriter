# 设定审核维度细则

> 逐项过。每条检查有明确 pass/fail 条件，不凭感觉。

---

## 一、世界观自洽性 (Worldview Consistency)

### 1.1 核心法则完整性
- [ ] 世界的根本规则是否已声明？（魔法来源 / 剧本系统 / 灵气根源）
- [ ] 核心法则是否有不可绕过的硬边界？
- [ ] 是否存在"法则为剧情服务被临时打破"的预设漏洞？

**pass 条件**：核心规则有一个明确的不可绕过声明。例："剧本不可被外部摧毁"、"灵力只来自灵根"。

### 1.2 物理/魔法法则一致性
- [ ] 已声明的规则在设定中是否前后一致？（例如第3节说"飞行消耗灵力"，第8节角色却说"飞了一整天不累"）
- [ ] 规则的特殊例外是否有解释？（"除了持有X道具"不算漏洞，是例外）

**pass 条件**：每条规则在设定各处引用时描述一致，例外有前置条件。

### 1.3 地理与历史逻辑链
- [ ] 设定的地点是否在历史/战争/资源分布上有关联？（不是随机命名堆砌）
- [ ] 历史事件是否有因果链？（A事件导致B事件，不是"同时发生了ACD"）
- [ ] 如果有"文明断层"（前文明覆灭），当前文明知道多少？不知道多少？（信息边界清晰）

**pass 条件**：至少一个地点有"为什么在那里"的解释，历史事件有因果链。

### 1.4 社会结构与信仰
- [ ] 社会阶层是否与力量体系对应？（强者=统治者合理，但需要解释"为什么强者愿意统治"）
- [ ] 主流信仰是否与核心法则关联？（如果法则可见，信仰通常不是"信不信"而是"怎么解读"）
- [ ] 是否存在"全民修仙但没人种地"的经济悖论？

**pass 条件**：社会阶层有对应的资源分配逻辑，信仰/意识形态与世界规则一致。

## 二、力量体系完整性 (Power System Completeness)

### 2.1 境界/层级定义
- [ ] 每个境界是否有可感知的差异？（战力参照、社会地位、视觉效果）
- [ ] 境界之间是否有小境界？
- [ ] 每个小境界是否有明确的含义或变化？

**pass 条件**：至少给出境界链和每个境界的核心能力/社会对应。小境界可选但建议有。

### 2.2 晋级条件与代价
- [ ] 晋级条件是否可操作？（不是"努力就行"，而是"需要经历X或完成Y"）
- [ ] 晋级失败是否有代价？
- [ ] 晋级是否可逆？（境界跌落的条件）

**pass 条件**：每层晋级有明确触发条件，失败有代价。

### 2.3 越级规则
- [ ] 低层能否越级战胜高层？如果可以，条件是什么？（非常态化、有代价）
- [ ] 越级是否有上限？（越一级可，越两级极度罕见，越三级不可能）
- [ ] 越级战斗是否有可感知的后果？（引来纠错者 / 消耗底牌 / 重伤）

**pass 条件**：越级边界清晰，不可常态化。

### 2.4 战斗规则
- [ ] 伤害与防御是否有逻辑？（属性克制 / 角色概念克制 / 层级差碾压）
- [ ] 克制关系是概念对概念还是属性对属性？
- [ ] 逃生/撤退是否有规则？（"打不过就瞬移跑"会废掉追杀线）

**pass 条件**：战斗有可预测的规则，逃生有条件。

### 2.5 体系漏洞
- [ ] 是否存在设定者已发现的漏洞？
- [ ] 漏洞是否有合理解释？（不是"这是bug请忽略"，而是"这是系统固有缺陷因为X"）
- [ ] 主角是否可以利用漏洞？反派是否可以利用？

**pass 条件**：体系漏洞已声明（如果有），利用方式有说明。

## 三、角色设定深度 (Character Depth)

### 3.1 欲望-缺陷-弧度对齐
- [ ] 主角欲望是否驱动故事？（不是被动接受任务）
- [ ] 主角缺陷是否会导致他付出代价？（不是"偶尔粗心"这种不会痛的缺陷）
- [ ] 成长弧是否与缺陷对应？（缺陷被修复的过程 = 弧线）
- [ ] 镜像反派是否共享主角的核心起点？（共享欲望/背景但选择相反路径）

**pass 条件**：欲望→缺陷→弧线→镜像反派形成完整链条。

### 3.2 OOC 警戒
- [ ] 是否列出了角色"绝不该做的事"？
- [ ] 警戒项是否可检测？（"不该杀人"可检测，"不该软弱"不可检测）
- [ ] 是否列出了需要提前铺垫的事？（为后续转变埋伏笔）

**pass 条件**：OOC 警戒有 2-3 条具体可检测的行为。

### 3.3 关系网完整性
- [ ] 主角与核心角色的关系是否都有定义？（盟友/对手/情感/债务）
- [ ] 多主角设定下，各主角是否有独立动机和互补冲突？
- [ ] 配角是否有独立于主角的动机？（不是"因为主角需要所以存在"）

**pass 条件**：核心关系网完整，配角有独立存在理由。

---

## 四、势力格局合理性 (Faction Logic)

### 4.1 各方动机
- [ ] 每个势力是否有独立于"服务剧情"的存在动机？
- [ ] 反派动机是否可理解？（不是"我就是坏"）
- [ ] 是否有势力间的关系矩阵？（敌对/中立/附庸/共生）
- [ ] 是否存在"某势力明明能轻松消灭另一方却长期共存"的不合理？

**pass 条件**：至少 3 个势力各有独立动机，关系矩阵明确。

### 4.2 组织层级
- [ ] 核心势力是否有内部层级？（领导者→执行者→外围）
- [ ] 晋升标准是否清晰？
- [ ] 是否存在内部对立/派系？

**pass 条件**：重要势力有 2-4 层内部结构。

### 4.3 反派分层
- [ ] 是否分小/中/大三层？
- [ ] 每层是否有独立目标、登场时机、与主角的关系？
- [ ] 三层之间是否有递进关系？（打完小的引出中的，打完中的暴露大的）
- [ ] 最终反派是否有镜像对抗设计？

**pass 条件**：反派分层完整，递进逻辑成立。

---

## 五、经济与资源系统 (Economy & Resources)

### 5.1 货币体系
- [ ] 是否有基础货币？是否有锚定物？（1灵石能买什么）
- [ ] 货币的产出方式是否稳定？（如果打怪掉落灵石，通货膨胀怎么办）
- [ ] 不同社会阶层的收入/消费差异是否合理？

**pass 条件**：货币有名称和锚定物，产出方式有基本逻辑。

### 5.2 稀缺性规则
- [ ] 关键资源（丹药/记忆碎片/命数币）的稀缺性是否有解释？
- [ ] 稀缺资源是否可再生？（如果不可再生，存量多少）
- [ ] 是否存在"主角无限供应但其他人找不到"的不合理？

**pass 条件**：至少 2 种关键资源的稀缺性有解释。

### 5.3 交易与流通
- [ ] 是否存在合法市场？是否存在黑市？
- [ ] 跨地区交易是否有规则？（灵石在A国和B国通用吗）
- [ ] 弃演者/边缘人群的经济来源是否有解释？

**pass 条件**：基础交易规则清晰。

---

## 六、时间线逻辑 (Timeline Logic)

### 6.1 关键节点
- [ ] 是否有历史年表？（至少 3-5 个关键事件）
- [ ] 事件之间是否有因果链？（不是随机时间点堆砌）
- [ ] "距今"的时间描述是否一致？

**pass 条件**：至少有建立事件、重大战争、当前状态三个节点的因果链。

### 6.2 时间逻辑
- [ ] 角色年龄与历史事件是否对齐？（角色说"100年前的事我很清楚"但他才25岁）
- [ ] 技术/社会发展速度是否合理？（中世纪文明持续了千年但最近十年突然进步了？如果有，是否解释？）

**pass 条件**：关键角色年龄与历史事件时间线对齐。

---

## 七、创意约束落地 (Creative Constraint Implementation)

### 7.1 反套路规则
- [ ] idea_bank.json 中声明的反套路规则是否在设定文件中有体现？
- [ ] 反套路规则是否在力量体系/世界观中有具体的机制支撑？（不是空话）
- [ ] 主角行为是否与反套路规则一致？（规则说"正确选择导致负面后果"，设定是否写出了具体例子）

**pass 条件**：每条反套路规则在设定集中有至少一处具体落地。

### 7.2 硬约束
- [ ] 硬约束是否在设定集中有对应规则？（不是只在 idea_bank 中提一句）
- [ ] 硬约束是否不可绕过？（如果可绕过，条件是什么）
- [ ] 硬约束是否会导致主角行动空间过窄？（约束到什么都做不了 → 故事无法推进）

**pass 条件**：每条硬约束在力量体系/世界观中有不可绕过的说明。

### 7.3 卖点与差异化
- [ ] 核心卖点是否在设定集中有独特体现？（不是"传统玄幻+系统流"）
- [ ] 差异化是否可执行？（不是"比别的书写得好"，而是"本作的X机制决定了Y"）
- [ ] 卖点是否会因后续设定而被冲淡？（例：卖点是"无名印记不给战力"，但后期给了战力）

**pass 条件**：核心卖点在设定集中有 1-2 处具体机制支撑。

---

## 八、行文约束与设定一致性 (Style/Setting Alignment)

### 8.1 情绪基调与世界观匹配
- [ ] 行文约束集的情绪温度区是否与世界观调性一致？（世界观是暖伤→行文是暖伤，不是世界观温暖但行文要求黑暗）
- [ ] MASTER_SETTING.json 的调性标注是否与行文约束一致？

**pass 条件**：行文情绪区间与世界观调性不冲突。

### 8.2 审查标准与设定规则匹配
- [ ] 行文约束中的审查条目是否会与设定矛盾？
  - 例：设定有大量内心独白机制，但行文约束禁止心理描写 → 冲突
  - 例：设定角色话少，但行文要求每章大量对话 → 一致
- [ ] 行文约束的禁止词列表是否会封死设定中需要的表达？

**pass 条件**：行文约束与设定规则无明显冲突。
