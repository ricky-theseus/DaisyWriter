# Changelog

## v0.1.0

首个可用版本，已具备以下能力：

- 解析本地 Markdown 章节文件
- 自动拆分章节号与章节标题
- 通过 Playwright + CDP 接管浏览器会话
- 自动填写番茄作者后台章节编辑器
- 单章立即发布
- 批量立即发布
- 使用番茄后台原生定时发布创建待发布章节
- 发布后跳转章节管理页进行状态校验
- 加入疑似单日 5 万字发布上限的保护阈值
- 记录“发布时间前 30 分钟内可能无法修改定时章节”的平台限制

当前仍属于可实际使用的原型版本，后续仍可继续增强：

- 更稳的章节管理页状态解析
- 更完整的异常弹窗处理
- 对已排定时章节修改流程的支持
- 更通用的配置能力
