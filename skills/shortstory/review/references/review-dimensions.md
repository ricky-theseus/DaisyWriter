# 短中篇审查维度定义

> 两套标准：**单章审**（循环一）和 **全篇终审**（循环二），子 agent 按调用场景切换。
> 评分和阻断分离：阻断是硬红线，不因总分高赦免。

## 字数分流

| 分类 | 范围 | 适用场景 |
|------|:----:|----------|
| 短篇 | 8000–29999 | 知乎盐选、番茄短篇 |
| 中篇 | 30000–50000 | 七猫付费中篇、出版向中篇 |

---

## 阻断判定（两阶段共用）

以下任意一条成立 → blocking = true，不参与定稿：

| 编号 | 阻断条件 | 适用阶段 |
|:----:|----------|:--------:|
| 1 | 前 500 字无有效钩子 | 两阶段 |
| 2 | 短篇 <8000 字；中篇 <30000 或 >50000 字 | 两阶段 |
| 3 | 平均句长 <18 或 >55 | 两阶段 |
| 4 | 极短句占比 >12% | 两阶段 |
| 5 | 连续同主语开头 >2 句 | 两阶段 |
| 6 | 存在情绪标签词（感到/觉得/一种/一阵） | 两阶段 |
| 7 | 副词修饰情绪 | 两阶段 |
| 8 | 结尾平铺直叙，无 twist 或情绪收束 | 两阶段 |
| 9 | 超过 1500 字无新信息推进 | 两阶段 |

> 阻断先判，通过后再算分。阻断未解决前即使单项满分也 blocking。

---

## 第一部分：单章审（循环一）

> 每次写完一章后 spawn 子 agent 做本章盲审。
> 评分对象是**当前章节的内容**，不是全文。

### 维度与权重

子 agent 对每项打 **0–10 分**，按权重折算百分制。

| 维度 | 权重 | 审查要点 |
|------|:----:|----------|
| 本章信息增量 | 0.30 | 这一章是否推进了主线？有无新冲突/新信息/人物变化？还是原地踏步？ |
| 文笔句法 | 0.25 | 句长是否在 25–50？极短句是否控制 ≤12%？有无情绪标签词和同主语开头问题？ |
| 章内节奏 | 0.20 | 本章内部有起伏吗？有张弛交替吗？还是从头平到尾？ |
| 章末钩子 | 0.15 | 结尾是否有钩子（悬疑/期待/悬念）吸引读者继续读下一章？ |
| 人设一致 | 0.10 | 本章人物行为是否符合前文已建立的人设？ |

### 维度详解

#### 1. 本章信息增量（0.30）

**判断依据**：
- 这一章提供了什么新信息？新冲突 / 新人物 / 新关系 / 新发现 / 人物变化
- 这些信息量是否够这一章的篇幅？还是大段篇幅只说了很少的事？
- 对比上一章：读者读完这一章知道了什么之前不知道的？

**参考标准**：
- 10分：重大推进（关键冲突爆发、核心反转、人物蜕变）
- 7–9分：有明确推进，信息量匹配篇幅
- 5–6分：少量推进，大部分篇幅在铺垫
- 0–4分：原地踏步，读完等于没读

#### 2. 文笔句法（0.25）

**判断依据**：
- 平均句长是否在 25–50 字范围内？极短句占比是否 ≤12%？
- 有无"感到/觉得/一种/一阵"情绪标签词？
- 有无"不是A而是B"折反句式？
- 连续同主语开头是否超过 2 句？
- 情绪传达是靠动作/对话/环境，还是靠标签词直白？

**参考标准**：
- 10分：句法成熟，情绪靠动作/对话传达，无标签词
- 7–9分：基本达标，1–2 处小瑕疵
- 5–6分：多处不达标，影响阅读体验
- 0–4分：严重句法问题，大量标签词

> 统计指标可由 `review_shortstory.py` 辅助提供，但判定以定性为准。

#### 3. 章内节奏（0.20）

**判断依据**：
- 这一章自身有没有起伏？快节奏段落和慢节奏段落是否交替？
- 有无超过 2 段连续慢节奏（纯叙述/心理）？
- 对话段落和叙述段落比例是否合理？

**参考标准**：
- 10分：张弛有度，节奏感强
- 7–9分：基本合理，节奏略显平但不出问题
- 5–6分：节奏单一，长时间无变化
- 0–4分：从头平到尾或节奏混乱

#### 4. 章末钩子（0.15）

**判断依据**：
- 本章结尾是否有钩子？悬念 / 疑问 / 危机升级 / 新发现 / 人物关系变化
- 这个钩子是否足够吸引读者点开下一章？
- 过渡章可以没有强钩子，但不可无结尾直接结束

**参考标准**：
- 10分：强悬念/冲突高潮，迫不及待想看下一章
- 7–9分：有明确钩子或自然过渡
- 5–6分：结尾平淡，无吸引点
- 0–4分：戛然而止或无结尾

#### 5. 人设一致（0.10）

**判断依据**：
- 本章人物的言行是否符合前文建立的性格和动机？
- 有无强行降智或 OOC 行为？

**参考标准**：
- 10分：完全一致，行为可基于性格预测
- 7–9分：基本一致，小处略有牵强
- 5–6分：1 处明显 OOC
- 0–4分：多人设崩塌

### 计算方式

```
单章总分 = Σ(维度得分 × 权重 × 10)
```

- 阻断先行，阻断通过后才算分
- 不设定稿门槛（单章不要求 95 分）
- 阻断 → 修 → 重审；零阻断 → start.py --advance

### 单章审输出格式

```json
{
  "stage": "chapter_review",
  "chapter": 3,
  "word_count": 2700,
  "status": "blocking" | "passed",
  "blocking": ["连续同主语开头超过2句"],
  "dimensions": {
    "本章信息增量": {"score": 8, "weight": 0.30, "note": "推进了新冲突但节奏略慢"},
    "文笔句法": {"score": 6, "weight": 0.25, "note": "句长21字偏低，2处情绪标签"},
    "章内节奏": {"score": 7, "weight": 0.20, "note": "前段紧张后段略平"},
    "章末钩子": {"score": 9, "weight": 0.15, "note": "悬念型结尾"},
    "人设一致": {"score": 9, "weight": 0.10, "note": "无问题"}
  },
  "suggestions": ["减少情绪标签词", "适当拉长句长"]
}
```

## 第二部分：全篇终审（循环二）

> 所有章节通过后做一次完整评审。评分对象是**整篇正文**。
> 定稿条件：零阻断 **且** 总分 ≥ 95。

### 维度与权重

子 agent 对每项打 **0–10 分**，按权重折算百分制。字数 <30000 走短篇表，≥30000 走中篇表。

#### 短篇表（<30000字）

| 维度 | 权重 | 审查要点 |
|------|:----:|----------|
| 开篇吸引力 | 0.28 | 前300–500字是否快速抛出冲突/危机/悬念？ |
| 情绪与反转密度 | 0.25 | 全文反转/爽点/虐点/爆点的数量和分布？ |
| 主线完整闭环 | 0.20 | 核心矛盾是否闭环？伏笔是否全部回收？ |
| 人设稳定 | 0.12 | 核心人物行为前后一致？有记忆点？ |
| 水文冗余 | 0.08 | 无关描写、重复心理、凑字数占比？ |
| 结局收束力 | 0.07 | 结尾是否满足情绪基调？有回味空间？ |

#### 中篇表（≥30000字）

| 维度 | 权重 | 与短篇的差异 |
|------|:----:|--------------|
| 开篇吸引力 | 0.22 | 中篇开篇节奏可稍缓 |
| 情绪与反转密度 | 0.23 | 中段可容纳更多层次反转 |
| 主线完整闭环 | 0.20 | 同上 |
| 人设稳定 | 0.15 | 中篇对人物弧光要求更高 |
| 水文冗余 | 0.10 | 中篇允许轻度过渡段落 |
| 结局收束力 | 0.10 | 中篇要求主题升华 |

### 维度详解

> 各维度评分标准与单章审类似，但评判范围从"本章"扩展到"全文"。

#### 1. 开篇吸引力

**判断依据**：
- 前300–500字是否有一个明确的"抓人点"？悬疑/情绪冲击/矛盾/金句/具体细节
- 前2000字是否让读者清楚"什么类型、主角是谁、核心冲突"？

**参考标准**：
- 10分：前300字内强冲突/悬念
- 7–9分：前500字有钩子，类型和主角明确
- 5–6分：有开篇但平淡
- 0–4分：慢热铺垫、背景堆砌、无抓人点

#### 2. 情绪与反转密度

**判断依据**：
- 全文有多少个情绪节点（反转/冲突爆发/虐心/爽点）？
- 分布是否均匀？有无超过1500字的长时间平淡？
- 短篇要求 ≥2–3 个节点，中篇 ≥4–6 个

**参考标准**：
- 10分：反转密集自然，情绪曲线有起伏
- 7–9分：节点充足，有一处明显高潮
- 5–6分：有反转但不够密集
- 0–4分：全程平铺

#### 3. 主线完整闭环

**判断依据**：
- 核心矛盾是否从头贯穿到尾？短篇一个，中篇可2–3层
- 伏笔是否全部回收？短篇不允许开放式留坑
- 转折是否有前文铺垫？

**参考标准**：
- 10分：完全闭环，伏笔全回收
- 7–9分：闭环，1–2小伏笔未回收但不影响
- 5–6分：勉强收尾，多个伏笔未回收
- 0–4分：核心矛盾未解决、烂尾

#### 4. 人设稳定

**判断依据**：
- 核心人物行为是否前后一致？有无强行降智？
- 短篇人物不需要复杂但必须一致
- 中篇要求人物有弧光（情绪变化/成长）

**参考标准**：
- 10分：人设统一，有记忆点
- 7–9分：基本一致，个别牵强
- 5–6分：1–2处明显 OOC
- 0–4分：多人设崩塌

#### 5. 水文冗余

**判断依据**：
- 无关景物描写、重复心理、凑字数对话占比
- 区分"氛围铺垫"和"无效灌水"：服务人物心境或主题的景物不算灌水

**参考标准**：
- 10分：无一字冗余
- 7–9分：少量可有可无但不影响整体
- 5–6分：10–20% 凑字数
- 0–4分：超 20% 无效内容

#### 6. 结局收束力

**判断依据**：
- 结尾类型：反转型 / 情绪收束型 / 开放式 / 循环型
- 是否匹配全文情绪基调？
- 禁止：戛然而止、机械降神、说教式总结、强行圆满

**参考标准**：
- 10分：回味悠长，读完有后劲
- 7–9分：完整收束，情绪落地
- 5–6分：完整但平淡
- 0–4分：烂尾、突兀、说教

### 计算方式

```
全篇总分 = Σ(维度得分 × 权重 × 10)
```

- 阻断先行，阻断通过后才算分
- 定稿条件：零阻断 **且** 总分 ≥ 95
- 分数映射星级：

| 星级 | 总分 | 说明 |
|:----:|:----:|------|
| 5星 | 85–100 | 爆款标准 |
| 4星 | 70–84 | 完整优质 |
| 3星 | 50–69 | 及格 |
| 2星 | 20–49 | 缺陷明显 |
| 1星 | 0–19 | 不合格 |

  "word_count": 13141,
  "status": "blocking" | "passed",

    "开篇吸引力": {"score": 9, "weight": 0.28, "note": "..."},

## 子 agent 注意事项

- 单章审和全篇终审是**两次独立的子 agent 调用**，不共用上下文
- 单章审不设通过分门槛，阻断消除即可通过
- 全篇终审必须零阻断 + ≥95 分才能定稿
- 阻断是硬红线：不因"总体还行"而放过
- 统计指标辅助（句长/极短句/情绪标签）可由 `review_shortstory.py` 提供，可选调用
