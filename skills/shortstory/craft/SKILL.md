# 短篇写作工艺约束

## 加载规则

- `shortstory-write`：Step 1 起草前必读本工艺标准
- `shortstory-review`：加载作为审查量化基准
- 任何 `shortstory-write` 或 `shortstory-batch` 会话均须加载本技能

## 质量门禁

| 指标 | 标准 | 阻断 |
|------|------|:---:|
| 开场钩子位置 | 前 500 字内 | >500 字无钩子 |
| 字数范围 | 8000-30000 | 超出 |
| avg_sentence_len | 25-50 字 | <18 或 >55 |
| 极短句占比 | ≤12% | >12% |
| 连续同主语开头 | ≤2 句 | >2 |
| 情绪标签词 | 0 | >0 |
| 副词修饰情绪 | 0 | >0 |
| 每 500 字信息推进 | ≥1 个新节点 | >1500 字无推进 |
| 结尾 twist/收束 | 必须有 | 平铺直叙结束 |

## 短篇与长篇的工艺差异

| 维度 | 长篇网文 | 短篇盐选 |
|------|---------|---------|
| 节奏 | 日更 4000 字，慢慢铺 | 单篇读完，每段都在抢时间 |
| 钩子 | 章末钩子保追读 | 前 500 字钩子定生死 |
| 人物 | 几十万字慢慢养 | 前 2000 字就要立起来 |
| 情节 | 多线并行，世界宏大 | 单线聚焦，一件事讲透 |
| 收束 | 留尾巴待续集 | 必须完整收束，留回味 |
| 对话 | 叙事主体 | 穿插提气用 |
