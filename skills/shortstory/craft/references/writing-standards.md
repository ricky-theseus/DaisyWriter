# 短篇工艺标准手册

> 知乎盐选向短篇专用强约束。与 `webnovel-craft` 互不覆盖，短篇的叙事逻辑和长篇完全不同。

---

## 一、句长控制

| 指标 | 标准 | 阻断 |
|------|:---:|:---:|
| 平均句长 | 25-50 字 | <18 或 >55 |
| 极短句占比(≤8字) | ≤12% | >12% |
| 长句占比(>30字) | ≥25% | — |

短篇阅读场景为"一次性读完"，句子节奏应比长篇更快。极短句用于爆发段落（紧张/惊讶/转折），长句用于氛围铺垫。

---

## 二、开场钩子

前 500 字必须做到至少一项：

- [ ] 抛出悬疑问题（"我穿越成师尊的白月光那天，她正站在诛仙台上。"）
- [ ] 制造情绪冲击（"我姐死的那天，全村都在放鞭炮。"）
- [ ] 设置矛盾场景（"婚礼进行到一半，新娘跑了。"）
- [ ] 金句开篇（"有些人的善良，是刻在骨子里的冷漠。"）
- [ ] 用具体细节抓人（动作/对话/环境细节，拒绝抽象概括）

前 2000 字（知乎回答前屏）必须让读者知道：**什么类型的故事、谁是主角、核心冲突是什么**。

---

## 三、节奏控制

### 段落间节奏

| 节奏型 | 长度 | 用途 |
|--------|:---:|------|
| 快 | 200-500 字 | 动作/对话/冲突爆发 |
| 中 | 500-1500 字 | 推进/铺垫/过渡 |
| 慢 | 1500-3000 字 | 氛围营造/心理刻画 |

- 连续快节奏段落 ≤ 3 段（防止读者疲劳）
- 连续慢节奏段落 ≤ 2 段（防止读者流失）
- **对话+动作** 与 **叙述+心理** 穿插比例为 1:1 到 2:1

### 全文节奏结构

```
开场(10%) → 发展(40%) → 转折(20%) → 高潮(20%) → 收束(10%)

开场   前屏抓人，告知类型+主角+冲突
发展   逐步释放信息，埋设伏笔
转折   关键反转（认知/关系/力量/身份）
高潮   矛盾爆发，情感顶点
收束   情绪落地，留回味或发人深省
```

## 四、悬念与伏笔

| 规则 | 说明 |
|------|------|
| 每篇至少 2 个伏笔 | 前 1/3 埋，后 1/3 收 |
| 伏笔显隐性 | 至少 1 个隐性（事后才意识到那是伏笔）|
| 不要"伏笔不回收" | 短篇篇幅有限，所有伏笔必须在篇内解决 |
| 转折必须在前铺垫 | 反转不是凭空出现，需读者细想能发现蛛丝马迹 |

---

## 五、人物塑造

短篇没时间慢慢养人物，必须**快速立像**：

### 方法
- **标签+行动**："陈默是那种开会从不说话的人——除非有人动他的键盘。"
- **对话立人**：用语气/用词/节奏刻画性格
- **关键时刻的选择**：一个抉择胜过三百字外貌描写
- **避免**：长篇累牍的身世背景介绍

### 标准
- 核心角色必须有 1 个记忆点（口头禅/习惯/反差）
- 配角 ≤ 3 个（短篇没空间养群像）
- 角色数量建议：2-4 个主要角色

## 六、结尾质量

| 类型 | 说明 | 适合 |
|------|------|------|
| 反转型 | 最后一句话推翻前文认知 | 悬疑/脑洞 |
| 情绪收束型 | 主角/读者的情绪得到释放 | 言情/古风 |
| 开放式 | 留白，读者自己品 | 文学性强 |
| 循环型 | 结尾回到开头场景，但心境已变 | 结构精巧 |

**禁止**：
- 戛然而止（该写的没写完）
- 机械降神（突然出现外力解决问题）
- 说教式总结（"这个故事告诉我们……"）
- 强行圆满（BE 也是好结局）

## 七、知乎盐选专项

### 投稿路径
| 方式 | 流程 | 周期 |
|------|------|:----:|
| 回答引流 | 在相关问题下发故事开头(2000字)→数据好→编辑找上门 | 1-7天 |
| 后台直投 | 登录盐选作者平台，上传完整稿件(≥8000字) | 20个工作日 |
| 编辑直投 | 通过知乎知学堂课程渠道，2000字可确定签约意向 | 最快 |

### 回答引流要点
- 选对问题：搜索热度高、问题本身有戏剧张力
- 开头 2000 字就是"试读"：必须包含钩子+类型定位+主角立像
- 前 3 天点赞量决定编辑注意度

### 收入预期
| 阶段 | 月收入 |
|------|:------:|
| 新人签约 | 2000-5000元/篇 |
| 稳定产出 | 5000-15000元/月 |
| 爆款 | 单篇过万 |
| IP改编 | 版权费 8000-200万 |

## 八、审查清单

用于写作后自检或 reviewer 检查：

- [ ] 前 500 字有钩子
- [ ] 前 2000 字明确类型/主角/冲突
- [ ] 字数 8000-30000
- [ ] 对话+动作占比 ≥ 叙述+心理占比
- [ ] 至少 2 个伏笔，全部回收
- [ ] 主角有 1 个记忆点
- [ ] 配角 ≤ 3 个
- [ ] 结尾有 twist 或情绪收束（非平铺结束）
- [ ] 无情绪标签词
- [ ] 无"不是A是B"折反句式
- [ ] 无说教式总结
- [ ] 平均句长 25-50，极短句 ≤12%
