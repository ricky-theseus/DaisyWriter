# 短篇钩子技术与质量评分

> 从 chinese-novelist + story-review 提取，适配短篇盐选标准。

## 前 20% 定生死

短篇的"前 20%"约等于前 1000-3000 字（视总字数）。在这段距离内必须建立钩子。

### 前 20% 必须达到的效果
1. **即时紧张感** — 读者立即感受到危机或冲突
2. **重大事件** — 发生推动剧情的重要事情
3. **情感冲击** — 读者产生强烈情绪（好奇、震惊、担忧）
4. **继续阅读的欲望** — 迫切想知道接下来发生什么

### 开头致命错误（绝对避免）
- 天气描写：与故事无关，读者无耐心
- 日常流程：无聊，没有冲突
- 缓慢铺垫：信息倾倒，没有行动
- 平淡对话：无张力，无意义
- 过度解释：讲述而非展示
- 回顾前文：短篇没有空间回顾

## 六种短篇强力开头技巧

### 1. 反常情境
呈现一个不符合常理的场景，激发好奇。
> 死人坐在办公桌前，正在写一份报告。
> 测谎仪显示她在说真话。但她说的事情，不可能发生在这个宇宙。

### 2. 震撼对话
用一句惊人的对话开场。
> "我把孩子卖了。"妻子平静地说，继续翻阅杂志。

### 3. 倒计时开场
从时间压力开始。
> 还有三分钟，炸弹就会爆炸。而拆弹包里少了一根红线。

### 4. 危机时刻
从角色面临最大危机开始。
> 门被踹开的瞬间，她知道藏不住了。

### 5. 重大发现
从发现关键线索或真相开始。
> 法医报告摊在桌上。死因是精心策划的谋杀。而嫌疑人只有一个——他自己。

### 6. 背叛/选择时刻
从信任崩塌或艰难决定开始。
> 救生艇只能载一个人。她的丈夫和女儿都在冰冷的海水里挣扎。

### 开头自测问题
如果对以下任何问题回答"否"，必须重写：
1. 读者读了前三段后，会想继续读吗？
2. 开头是否有冲突或危机？
3. 开头是否有意外或转折？
4. 开头是否让读者产生强烈情绪？
5. 开头是否避免了平淡的日常、天气、背景说明？

## 短篇的展示而非讲述

| 讲述（避免） | 展示（使用） |
|-------------|-------------|
| 他很愤怒 | 他握紧拳头，指节发白 |
| 她很美丽 | 他凝视着她，忘记说话 |
| 他很紧张 | 他反复调整领带 |
| 房间很乱 | 衣服扔在沙发上，外卖盒堆在桌上 |
| 她很伤心 | 她低头搅咖啡，勺子停了很久 |

## 短篇质量评分标准（80分制）

每篇交付前对照评分。>60 分可交付，>70 分优秀。

| 维度 | 评分 | 检查要点 |
|------|:----:|---------|
| 开头吸引力 | /10 | 前 3 段抓住读者？前 500 字有钩子？ |
| 情绪拉扯力 | /10 | 读者是否产生强烈情绪？有压抑→释放？ |
| 人物弧光 | /10 | 前 2000 字人物立起来？行为一致？ |
| 对话质量 | /10 | 对话推动情节？有潜台词？可区分角色？ |
| 反转/悬疑设置 | /10 | twist 质量？信息差利用？反转有铺垫？ |
| 节奏控制 | /10 | 张弛有度？信息密度波浪？段落呼吸？ |
| 信息密度 | /10 | 每 500 字有新推进？无注水段？ |
| 语言质量 | /10 | 无 AI 痕迹？白描为主？用词精确？ |

评分规则：
- 每一项对照检查清单打分，不凭感觉
- 如果某项低于 6 分，该维度必须修改后重新评分
- 总分低于 60 分：整篇需要大改
- 总分 60-70 分：基本可交付，打磨细节

## 短篇结尾类型

| 类型 | 适用场景 |
|------|---------|
| 反转结尾 | 悬疑、社会派——最后一句颠覆全文 |
| 情绪余韵 | 虐恋、治愈——留白，不写死 |
| 闭环收束 | 所有线索回收，形成完整故事圆环 |
| 开放式 | 留有回味空间，但不让读者觉得没写完 |
