# DaisyWriter for Codex CLI

Codex CLI support is in development.

## Planned Integration

Codex CLI uses `.codex/instructions/` for skill-like modules. We plan to provide:

- Instruction files for each writing domain
- Conversion scripts from OpenCode SKILL.md format
- Publishing workflow integration

Track progress: https://github.com/ricky-theseus/DaisyWriter/issues
