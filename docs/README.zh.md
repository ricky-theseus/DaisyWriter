## DaisyWriter 是什么？

DaisyWriter 是一个 AI 编程助手的技能合集，可以让你的 AI 助手变成一个全功能写作工作室。它提供了 **24 个可组合的技能**，覆盖网文、短篇、技术博文和发布自动化。

## 快速开始

```bash
git clone https://github.com/ricky-theseus/DaisyWriter.git
cd DaisyWriter
```

在 OpenCode 中加载一个技能：

```
/webnovel-init "我的修仙之旅"
```

## 技能一览

| 领域 | 数量 | 说明 |
|------|------|------|
| 📚 **网文创作** | 12 | 立项 → 规划 → 写章 → 审查 → 批量 → 仪表盘 |
| 📝 **盐选短篇** | 5 | 初始化 → 滚动写 → 盲审 → 工艺约束 → 拆解 |
| 💻 **技术博文** | 5 | 写作 → 拆解 → 批量 → CSDN 同步 → CSDN 上传 |
| 🤖 **番茄发布** | 1 | 浏览器自动化发布到番茄作者后台 |
| 🔧 **封面生成** | 1 | AI 生成 600×800 小说封面 |
